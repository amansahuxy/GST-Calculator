# 🧾 GST Calculator

A simple GST Calculator built using HTML and JavaScript. Users can enter an amount, choose a GST rate, and instantly calculate the GST amount and the final payable amount.

## 🚀 Features

* Enter the original amount
* Select GST rate (5%, 12%, 18%, or 28%)
* Calculate GST instantly
* Display GST amount
* Display total amount after GST
* Beginner-friendly project

## 🛠️ Technologies Used

* HTML5
* JavaScript

## ▶️ How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## 🧠 How It Works

1. Enter the original amount.
2. Select a GST rate from the dropdown.
3. Click the **Calculate GST** button.
4. The application calculates:

   * GST Amount
   * Total Amount (including GST)

### Formula Used

```text
GST Amount = (Amount × GST Rate) / 100

Total Amount = Amount + GST Amount
```
